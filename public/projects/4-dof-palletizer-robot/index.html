4-DOF Palletizer Robot

Team project to develop a 4-DOF palletizer robot for automatic parcel sorting and rack storage.

Associated with: Team Project (10 Members) | Project Date: Jan 2019 - May 2019

Project Overview

This project focused on building an automated parcel sorting and storage robot as a 10-member team effort.

The system used OCR to read parcel labels, classify destination/storage information, and execute automated placement onto target rack positions.

It was a cross-course class project that integrated knowledge from multiple subjects, including robot kinematics, artificial intelligence, and mechanical design. The team developed both hardware and software from scratch under tight budget constraints, with strong emphasis on practical design trade-offs, reuse of available parts, and reliable system integration.

Objectives

  • Build a full workflow for parcel intake, OCR-based label recognition, and rack assignment.
  • Develop a robotic handling subsystem to pick and place parcels into storage racks.
  • Achieve stable and smooth simultaneous motion control for all arm joints during operation.
  • Validate integrated operation from label reading to final rack placement.

My Contribution

I contributed mainly on robotic hardware and control bring-up for the palletizing arm.

  • Hardware design: Designed the mechanical hardware concept for a 4-DOF palletizer arm.
  • Component sourcing: Led sourcing and selection of key arm components for build feasibility.
  • Controller debugging: Participated in debugging and tuning the controller for smooth simultaneous 4-DOF motion.
  • Integration support: Worked with the team to connect arm behavior with the OCR-driven sorting logic.

Outcome

  • Delivered an integrated prototype that could read parcel labels and automate rack storage.
  • Established workable hardware architecture and sourced components for the 4-DOF arm.
  • Improved coordinated joint control performance through controller debugging and tuning.
  • Demonstrated end-to-end engineering execution across kinematics, AI, controls, and mechanical design within a limited-budget class-project environment.
